# Lando Drupal 8 contribtution setup



## Prerequisites
1. Download and install lando from https://docs.lando.dev/basics/installation.html#preflight-checks
2. Install Composer from https://getcomposer.org/doc/00-intro.md#installation-linux-unix-macos
 
## Local Setup instructions:
Run following command and you will be asked for inputs about the drupal core branch and 
installation profile.
```
./install.sh
```
